# Idea del juego

Simon sera una mascota virtual musical.

## Concepto inicial

- Simon puede animarse, cantar e interactuar.
- El primer juego planeado sera una memoria musical estilo Simon Says.
- La pantalla touch permitira tocar zonas, botones o elementos musicales.
- El audio y la bocina se definiran en una etapa posterior.

## Fase actual

En esta primera fase no se programa la logica del juego.

El enfoque v0.1 es preparar:

- pantalla
- touch
- base del proyecto
- documentacion
- scripts de diagnostico
