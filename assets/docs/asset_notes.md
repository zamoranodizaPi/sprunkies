# Sprunkies asset pack v0.1

Assets originales para el primer prototipo de Simon.

Resolución objetivo: 480x320 landscape.

Incluye:
- Fondo de día de campo.
- Nubes.
- Notas musicales.
- Sprites de Simon: idle, blink, sing, happy.
- Audios WAV cortos para botones, canto, éxito y error suave.

Estos assets son placeholders bonitos y ligeros para arrancar el desarrollo en Raspberry Pi 3 con Pygame CE.
